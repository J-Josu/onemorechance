## Description 📝

Based on the H.D.P game (Hasta donde puedas) omc is a boardgame made in the browser, with multiplayer support and mobile first.

## Stack 🛠

- Sveltekit
- Typescript
- Daisyui
- Tailwindcss
- Websockets
- AWS
- Docker
- Github Actions
- Ansible
- Terraform
